If you wish to add a reward token for distribution to liquidity providers, please follow the process below that matches your circumstances.

### Your pool already exists but your token is not on the Orca Token List.

* See [here](/token-creators/eclipse-orca-token-list) to request your token is added to the *Orca Token List.* **Ensure you provide contact details**.
* Await confirmation that the token has been added to the *Token List,* the warning triangle in the UI will disappear.
* Await confirmation that the rewards can be accommodated by Orca.
* Submit reward tokens to Orca using the address Orca provides to you
* Await confirmation that the rewards are live.

* Create your pool, if you are an advanced user refer to [How to create an initial pool for an asset](/token-creators/clmm-pools) otherwise follow the [Creating a Pool Tutorial](/liquidity/beginner-guide/classic-community-listing/creating-a-pool-tutorial).
